# DJ Casio Photo Gallery

## Directory Structure

### `/headshots`
Professional headshot photos suitable for:
- Press kits
- Event promotions
- Social media profiles
- Print materials

**Specifications:**
- Resolution: 300 DPI (minimum)
- Format: JPG, PNG
- Recommended sizes: 1500x2000px or higher

### `/performances`
Action shots and performance photos from:
- Live events
- Festival appearances
- Club performances
- Studio sessions

**Usage Rights:** All photos include proper credits and usage rights

### `/backgrounds`
Promotional background images for:
- Flyers and posters
- Social media headers
- Website banners
- Event promotions

---

## Adding Photos

To add new photos:
1. Ensure photos meet the specifications above
2. Place in the appropriate subdirectory
3. Include a `credits.txt` file with photographer name and rights
4. Update this README with new additions
